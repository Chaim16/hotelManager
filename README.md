# hotelManager
酒店管理系统

天下集团酒店管理系统，包含入住管理、会员管理、客房管理、订单管理四大模块
所用的技术栈为HTML、CSS、JavaScript、JQuery、JSTL、SpringBoot、MyBatis、java、MySQL、数据库连接池、JSP、XML
